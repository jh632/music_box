### 详细分配表

| GPIO | 外设 | 引脚 | 功能 | 说明 |
|------|------|------|------|------|
| **0** | Boot 按键 | KEY | 切歌 | 无上拉 |
| **—** | MAX98357A | VIN | 供电 (3.7~5V) | 接 5V 电源，模块输入 |
| **—** | MAX98357A | GND | 地 | 接 GND |
| **16** | MAX98357A | BCLK | I2S 位时钟 | 与 17/18 相邻，便于 PCB 走线 |
| **17** | MAX98357A | LRC | I2S 左右声道时钟 | WS/LRCLK |
| **18** | MAX98357A | DIN | I2S 数据输入 | 音频数据流 |
| **15** | MAX98357A | SD | 关断控制 (低有效) | GPIO 输出，默认拉高使能，可软件静音/省电 |
| **—** | MAX98357A | GAIN | 增益选择 | PCB 硬接 GND → **9dB** (4Ω 喇叭够用)；也可接 VDD → 15dB 或加跳线选择 |
| **48** | WS2812B | DATA | LED 数据线 | RMT 外设驱动 |
| **1** | 光敏电阻 | AO | 环境光检测 | ADC1_CH0，12bit 采样，调节 LED 亮度 |
| **—** | 光敏电阻 | VCC/GND | 供电 | 接 3.3V/GND，DO 悬空不用 |
| **8** | OLED | SDA | I2C 数据 | I2C0，标准开漏 + 外部上拉 4.7kΩ |
| **9** | OLED | SCL | I2C 时钟 | 同上 |
| **—** | OLED | VCC/GND | 供电 | 接 3.3V/GND |

### 设计要点

1. **GPIO16/17/18 连续排列** — I2S 三线信号在 PCB 上可平行走线，减少串扰，Layout 友好
2. **GPIO15 紧邻 I2S 组** — SD 关断线方向与 I2S 一致，方便与功放模块一起走线
3. **GAIN 硬接 GND (9dB)** — 搭配 3W/4Ω 喇叭，9dB 增益在 3.3V I2S 信号下输出约 2~3W，够用；如需更大音量可改接 VDD (15dB) 或预留 0Ω 跳线
4. **WS2812B 供电** — 板载
5. **光敏电阻电路** — AO 接 GPIO1 (ADC1_CH0)，建议 10kΩ 下拉电阻与光敏电阻串联接地形成分压，亮→阻值↓→电压↑
6. **OLED 上拉** — GPIO8/9 内部上拉约 45kΩ，I2C 标准速率下应外加 4.7kΩ 上拉到 3.3V
7. **未使用 GPIO 充裕** — 后续如需添加按键、旋转编码器、SD 卡等，有余量
8. **板载按键 boot** — GPIO0

### 已占用的关键引脚（不可用于外设）

| GPIO | 占用原因 |
|------|----------|
| GPIO0 | Boot 按键（低电平进入下载模式） |
| GPIO3 | JTAG 控制 |
| GPIO19/20 | 原生 USB D-/D+（若用 USB CDC） |
| GPIO26-32 | 内部 Flash (N8R8，不外引) |
| GPIO33-37 | 内部 Octal PSRAM (N8R8，不外引) |
| GPIO43/44 | UART0 TX/RX（USB-UART 桥接） |
| GPIO45/46 | 下载模式相关，尽量不用 |
